# HR_Analytics_Powerbi

📌 Project Overview

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, workforce demographics, and key HR metrics. The dashboard enables HR professionals to monitor workforce trends and make data-driven decisions to improve employee retention and organizational performance.

---

## 🎯 Objectives

- Analyze employee attrition trends.
- Monitor workforce demographics.
- Identify factors influencing employee turnover.
- Evaluate job satisfaction across departments.
- Track employee performance using interactive KPIs.
